## Tech Stack & Features

- **HTML5:** Semantic architecture structuring the Home, About, and Contact layouts.
- **CSS3:** Modern, responsive "Tech Minimalist" custom stylesheets utilizing modern CSS reset guidelines, custom form focus states, and flexbox design layouts.
- **GitHub Pages:** Cloud hosting solution utilizing automated deployment branches directly from source.

## Contributing

We welcome contributions from all group members. Please follow the standard GitHub workflow for contributing:

1. Fork the repository and create a new branch for your feature or bug fix.
2. Make your changes and commit them with clear, descriptive messages.
3. Push your changes to your forked repository.
4. Open a Pull Request to the main repository for review and merging.
5. Ensure that your code adheres to the project's coding standards and guidelines.
6. Participate in code reviews and provide constructive feedback to your peers.
7. Communicate effectively with the team to ensure smooth collaboration and project progress.

```
git clone
https://github.com/shekinahwebdev/cloud-lab-group-project.git

```

### Pull latest changes from the team

git pull origin main

### Make your updates inside code files, then stage changes

git add .

### Commit with a clear description of your update

git commit -m "Added [Your Name] bio information to about.html"

### Push to the cloud repository

git push origin main
